# API Overview

> FastAPI endpoint summary.

## Health

| Method | Path | Purpose |
|--------|------|---------|
| `GET` | `/health` | Liveness probe + version |
| `GET` | `/version` | Application version |
| `GET` | `/health/slack-token` | Slack token status |
| `POST` | `/health/slack-token/exchange` | Exchange Slack token |
| `POST` | `/health/slack-token/refresh` | Refresh Slack token |

## PRD Flow

| Method | Path | Purpose |
|--------|------|---------|
| `POST` | `/flow/prd/kickoff` | Start a PRD generation flow |
| `POST` | `/flow/prd/approve` | Approve current iteration |
| `POST` | `/flow/prd/pause` | Pause a running flow |
| `POST` | `/flow/prd/resume` | Resume a paused flow |
| `GET` | `/flow/prd/resumable` | List resumable runs |
| `GET` | `/flow/prd/runs` | List all runs |
| `GET` | `/flow/prd/runs/{run_id}` | Get run details |
| `GET` | `/flow/prd/jobs` | List all crew jobs |

## Slack

| Method | Path | Purpose |
|--------|------|---------|
| `POST` | `/slack/kickoff` | Start PRD flow from Slack |
| `POST` | `/slack/events` | Events API webhook |
| `POST` | `/slack/interactions` | Block Kit button clicks |
| `GET` | `/slack/oauth/callback` | OAuth v2 callback |

## Publishing

| Method | Path | Purpose |
|--------|------|---------|
| Various | `/publishing/*` | 10 automation endpoints for Confluence/Jira delivery |
